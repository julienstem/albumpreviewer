# Album Previewer

This website was made to allow building and managing an album.
You can load tracks, details and covers to select and reorder them.
You will have real time information such as number of tracks or duration of the whole album.
You can export and import the album allowing you to share it.

## Local

If you want to run this locally you can use either the docker or npm/yarn

**Docker**
`
docker compose up
`

**npm**
`
npm run dev
`

**yarn**
`yarn dev`
